# Chrome Web Store 公開用情報

## 拡張機能名
Zenn Scrap TOC - 目次自動生成

## 簡単な説明（132文字以内）
ZennのScrapページに自動的に目次を表示。記事のように快適にScrapを読めます。

## 詳細な説明（16,384文字以内）
ZennのScrapページに、Articleページと同様の目次機能を追加するChrome拡張機能です。

### 主な機能
📑 **自動目次生成**
- Scrapページの見出し（H1〜H3）を自動検出
- 階層構造を保持した目次を生成

🎯 **スムーススクロール**
- 目次項目をクリックで該当セクションへ移動
- なめらかなスクロールアニメーション

🔍 **スクロール連動**
- 現在読んでいるセクションを自動ハイライト
- 長いScrapでも現在位置が一目でわかる

### 使い方
1. 拡張機能をインストール
2. ZennのScrapページを開く
3. 自動的に目次が表示されます
4. 目次をクリックして快適に読書

### プライバシー
- ユーザーデータの収集なし
- 外部サーバーとの通信なし
- ローカルストレージのみ使用（設定保存用）
